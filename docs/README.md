# Dokumentáció (docs/)

## Tartalom
- `sprint-01/` – piackutatás, interjúk, ADR-ek, AI napló (az eredeti template szerinti struktúrában)
- `sprint-02/` – specifikáció, wireframe-ek, teszt riportok és egyéb sprint 2 artefaktumok
- `submission/` – beadáshoz összerakott, röviden hivatkozható összefoglalók
- `template/` – a kurzus eredeti README / template anyagai (referenciának)

A beadási szempontból legfontosabb belépő pont: `docs/submission/`.
